bool QUrl::cdUp ()

Use resolved("..") instead.

For example, if you have code like

 QUrl url("http://www.trolltech.com/Developer/");
 url.cdUp();

you can rewrite it as

 QUrl url("http://www.trolltech.com/Developer/");
 url = url.resolved("..");
